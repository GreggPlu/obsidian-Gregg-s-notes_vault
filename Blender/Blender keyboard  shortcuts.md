shift A - add object
shift X - delete object

TRANSFORMS:
G - grab (to move)     Alt-G clear transforms (moves it back to starting position)
R - rotate                     Alt-R clears transform to original rotation
S - scale                      Alt-S clears transform to original scale

while doing these transforms, picking
X limits to x-axis
Y limits to y-axis
Z limits to z-axis

CTRL A - apply transformations

N - pulls out the menu from the right. select Item tab for object properties

shift right mouse button - moves 3D cursor

shift D - creates duplicate object
ALT D - creates a new instance of an object

TAB - toggle between Object and Edit modes
